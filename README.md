# social-media-content-calendar — Structured, exportable social media calendars

> Generate complete content calendars from a single brand description. Platform-specific posts, hashtags, image prompts, and CTAs — ready to export to Buffer, Hootsuite, or Later.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
social-media-content-calendar builds a full content calendar for any brand or niche. It starts with brand and audience analysis — voice, target demographic, and 3–5 content pillars — then produces platform-specific daily posts with hashtags, image prompts, and CTAs. Output is structured for immediate export to major schedulers.

## Features
- **Platforms** — Instagram, Twitter, LinkedIn, TikTok, Facebook, YouTube, Threads
- **Timeframe** — default 4 weeks, configurable
- **Brand analysis** — voice, target audience, 3–5 content pillars
- **Per-day posts** — platform-optimized copy
- **Extras** — hashtags, image prompts, CTAs
- **Export-ready** — Buffer / Hootsuite / Later compatible

## Usage / Quick Start
```
/social-media-content-calendar "eco-friendly coffee brand" "instagram,twitter,linkedin" "4 weeks"
```
Arguments: brand/niche description · platforms (comma-separated) · timeframe
